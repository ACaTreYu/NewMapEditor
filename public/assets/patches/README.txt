Graphics Patch Folders
=====================

Each subfolder here is a "patch" containing game graphics.

A patch folder should contain:
  imgTiles.png    - Tileset (640px wide, 16x16 tiles, 40 per row)
  imgFarplane.png - Space background (shown behind empty tiles)
  imgTuna.png     - HUD/overlay layer

Supported formats: .png, .jpg, .bmp, .gif

To use a patch: click the folder icon in the Tileset panel title bar
and select the patch folder.
