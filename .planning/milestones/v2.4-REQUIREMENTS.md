## v2.4 Requirements

Requirements for MDI Window Controls milestone. Child map windows get full window management.

### Window Controls

- [x] **CTRL-01**: User can minimize a child window via minimize button in title bar
- [x] **CTRL-02**: User can maximize a child window via maximize button in title bar
- [x] **CTRL-03**: User can restore a minimized window by clicking its bar at workspace top
- [x] **CTRL-04**: User can restore a maximized window via restore button (replaces maximize button)
- [x] **CTRL-05**: User can close a child window via close button (existing — no change)

### Minimize Behavior

- [x] **MINZ-01**: Minimized window appears as a compact titled bar at top of workspace
- [x] **MINZ-02**: Multiple minimized windows stack horizontally without overlapping
- [x] **MINZ-03**: Minimized window bar shows document name

### Maximize Behavior

- [x] **MAXZ-01**: Maximized window fills the entire MDI workspace area
- [x] **MAXZ-02**: Maximized window hides its title bar for maximum canvas space
- [x] **MAXZ-03**: Maximized window canvas resizes to fill available space
- [x] **MAXZ-04**: Double-click title bar toggles maximize/restore
